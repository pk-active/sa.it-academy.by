# 05. Ansible start
### Install Ansible on CentOS
```bash
yum install -y epel-release
yum install -y ansible
```


#### Setup existing local user to run commands without password:
```bash
adduser ansible
passwd ansible <<EOF
ansible
ansible
EOF
echo "ansible ALL=(ALL) NOPASSWD: /usr/bin/yum update" >> /etc/sudoers
tee -a /etc/ansible/hosts <<EOF
[localhost]
127.0.0.1 ansible_user=ansible
EOF
```

#### Add ssh-key to jump host "Bastion":
```bash
ssh-keygen -t rsa -b 2048
ssh-copy-id -i ~/.ssh/id_rsa.pub jump_sa@178.124.206.48
```
#### Create users and add ssh-key to current hosts:
##### CentOS host:
```bash
ssh jump_sa@178.124.206.48
ssh root@192.168.200.21
adduser sa-centos-21
passwd sa-centos-21
yum install sudo
echo "sa-centos-21 ALL=(ALL) NOPASSWD: /usr/bin/yum -y update" >> /etc/sudoers
exit
ssh-copy-id -i  ~/.ssh/id_rsa.pub sa-centos-21@192.168.200.21
```
##### Ubuntu host:
```bash
ssh jump_sa@178.124.206.48
ssh root@192.168.200.22
adduser sa-ubuntu-22
echo "sa-ubuntu-22 ALL=(ALL) NOPASSWD: /usr/bin/apt update, /usr/bin/apt -y dist-upgrade" >> /etc/sudoers
exit
ssh-copy-id -i  ~/.ssh/id_rsa.pub sa-ubuntu-22@192.168.200.22
```
#### Prepare bastion ssh_config
```bash
Host bastion
        User jump_sa
        HostName 178.124.206.48
Host 192.168.200.*
        ProxyJump bastion
```
